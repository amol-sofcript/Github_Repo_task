# Github_Trending_Repositories in Android - Coding Assignment

The main idea of the test project is to build a Native Android App that has 2 Screens:

A list of trending Github repositories of android

A simple “Detail View” of the detail information of the repository, been accessed when the user taps on one of the repositories of the list.

Implement a background sync service which will fetch data from Github API after every 15 minutes and store it in a local database.

Users can check all repositories offline also.

If the device is restarted then background service of the app will start automatically.

Use the following:

Retrofit

Kotlin

Dagger
Coroutines or RxJava
Room DB
MVVM Architecture Pattern
